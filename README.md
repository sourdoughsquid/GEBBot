# GEBBot
Generic Economy and Blackjack Bot for discord.py

This was made custom for an old discord of mine, so a LOT of the messages will only make sense for that server, but you can change anything you please. You can do whatever you want, but if you make something with this then letting me know what it is would be so cool, just to see what I've contributed to :D

# Default Commands 
(all commands have a slash beforehand)

Allin- gambles ALL your balance, with choosable odds. 
Blackjack- Generic game of blackjack, use !hit or !stand during a game.
Daily- a daily reward, that increases as your streak gets higher (caps at 5).
GetAJob- (cooldown of an hour by default) get employed, random chances of how much money you get (or lose)
Give- gives yourself money. This requires to put your discord user ID in line 86, or wherever the give command code is stored.

# Requirements
-discord.py ofc
-putting your discord bot token at the bottom of the file.
